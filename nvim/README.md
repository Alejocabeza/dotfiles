# NEOVIM SETUP

Esta es la configuracion de mi editor favorito como lo es `NEOVIM`, este es un editor de terminal que tiene una gran variedad de funcionalidades y es muy recomendable para trabajar en proyectos de desarrollo. Para ver la configuracion de `neovim`, este editor es tan customizable como tu lo quieras hacer, por mi parte me gusta que sea minimalista y solamente tenga lo necesario para poder trabajar multiples lenguages y tecnologias. para esto no me preocupo mucho por lo que es el apartado visual del editor si no que me centro en la funcionalidad y la velocidad de desarrollo.
